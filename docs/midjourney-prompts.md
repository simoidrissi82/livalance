## Tips for Best Results

1. **For Hero Background**: Use Option 1 or 3 - they provide the softest, most versatile backgrounds
2. **Image Specifications**: 
   - Aspect Ratio: 16:9 (perfect for web hero sections)
   - Resolution: Request 4K for crisp display on retina screens
   - Style: Use `--style raw` for more control and less artistic interpretation
3. **Modifications**: 
   - Add `--chaos 10` for more variation
   - Add `--stylize 250` for more artistic interpretation
   - Add `--weird 100` for unique compositions
4. **Upscaling**: After generation, use "Upscale (4x)" for final hero image

## Usage Guidelines

- **Desktop Hero**: 1920x1080px minimum
- **Mobile Hero**: Ensure key elements are center-focused
- **Overlay**: These backgrounds work best with text overlay using dark text (#0B1220) or white text with subtle shadow
- **Opacity**: Consider using at 80-90% opacity with a solid color behind for better text readability
